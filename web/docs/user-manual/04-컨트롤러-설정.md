# 4. 컨트롤러 설정

단건 컨트롤러의 알람·설정온도·편차를 보거나 적용합니다. 목록의 **설정** 모드 또는 카드의 설정 버튼으로 엽니다.

## 목록 — 설정 모드

![목록 설정 모드](./images/04-list-settings-mode.png)

### 이 화면에서 할 수 있는 것

- **설정 모드 탭**: 각 카드에 알람·설정온도 요약과 **기본값** / **적용**이 보입니다.
- **알람 / 설정온도 · 편차 접기**: 요약 줄을 눌러 상세 슬라이더를 펼칩니다.
- **기본값**: 입력 중인 변경을 되돌립니다.
- **적용**: 변경이 있을 때만 활성화됩니다. 전송 후 ACK·현장 반영 피드백을 확인합니다.
- **오프라인**: 값이 `—`이거나 적용이 막힐 수 있습니다. 연결 상태를 먼저 확인하세요.

## 설정 패널 (펼침)

![설정 패널](./images/04-list-settings-panels.png)

![설정 펼침 예시](./images/04-settings-expanded.png)

### 이 화면에서 할 수 있는 것

- **온도·습도 게이지**: 현재값과 허용 범위를 한눈에 봅니다.
- **채널 A / B / C**: 채널별 현재값(해당 시).
- **알람**: 온·습 상·하한.
- **설정온도 · 편차**: 설정온도, 편차, 환기 최저·최고.
- **조회 전용**(뷰어): 상단에 조회 전용 안내가 뜨고 슬라이더·적용이 잠깁니다.

> PC는 카드 인라인 패널, 모바일은 bottom sheet입니다. → [08-모바일.md](./08-모바일.md)
